# MyShop

A new Flutter project.

## Getting Started

1. Download and install android studio.
2. run android sdk.
3. run the lib/main code. 

## Features

1. State Management with provider
2. valid login
3. Instant change
4. add/update/delete product
5. image picker
6. profile
7. form
8. Http requests
9. storing product in firebase DB
10. Animations
11. refresh Loader
